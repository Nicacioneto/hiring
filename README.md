# Transação de BTC Testnet via API Block.io

## Pré Requisitos

	- Rails: >5.1
	- Ruby: >2.4

## Configs Iniciais

Puxando depêndencias do projeto:
```sh
$ bundle install
```

Iniciando servidor:
```sh
$ rails s
```

Aplique em sua URL:
```sh
localhost:3000/transactions/new
```

### Testes

Realizando testes:
```sh
$ rake test
```
